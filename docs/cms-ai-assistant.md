# دستیار AI — حالت Inline (Notion-style)

## نحوه استفاده

1. بخشی از متن را **انتخاب** کنید
2. منوی تیره **Bubble** بالای انتخاب ظاهر می‌شود
3. عملیات را بزنید (بهبود، سئو، لحن آرایه، …)
4. **Review** قبل/بعد را ببینید
5. **Push** برای اعمال یا **رد** برای کنسل

## انتخاب مدل

در toolbar ویرایشگر، کنار آیکون ✨ مدل را انتخاب کنید (ذخیره در localStorage).

API مدل‌ها: `GET /api/admin/blog/ai/models`

## عملیات inline

| دکمه | کار |
|------|-----|
| بهبود نوشتار | بازنویسی فارسی روان |
| ادامه بده | ادامه از محل کرسر |
| کوتاه‌تر / گسترش | ویرایش طول |
| سئو کن | بهینه inline برای سئو |
| لحن آرایه | مطابق چارچوب برند آرایه |

## SEO و چارچوب

- تب **SEO** در سایدبار: فیلدها + SEO Guard
- چارچوب محتوا: `lib/cms/ai/araayeFramework.ts`

- همه درخواست‌ها server-side
