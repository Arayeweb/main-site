# راهنمای ویرایشگر بلاگ (فارسی)

## شروع کار

1. `/admin/manager/blog` → **مقاله جدید**
2. عنوان و متن را در Tiptap بنویسید (autosave هر ۲.۵ ثانیه)
3. تب **سازماندهی** → دسته و نویسنده
4. تب **SEO** → کلمه کلیدی + بررسی SEO Guard
5. تب **AI** → بریف و outline قبل از نوشتن بخش‌ها

## Workflow توصیه‌شده

```
کلمه + intent → بریف AI → outline → نوشتن بخش‌ها → SEO → preview → بررسی → انتشار
```

## AI

- همیشه پیش‌نمایش را ببینید
- آمار بدون منبع را تأیید نکنید
- محتوای پزشکی نیاز به بازبین انسانی دارد

## Preview

دکمه پیش‌نمایش → لینک موقت ۱۵ دقیقه‌ای، noindex

## انتشار

فقط وقتی SEO Guard «بدون خطای بحرانی» نشان دهد
